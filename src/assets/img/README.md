# Assets Directory

This directory is for your blog images and media files.

## Profile Image

Add your profile image here as `profile.webp` (or update the path in `config.py` to match your filename).

## Blog Post Images

Create subdirectories for different blog posts or topics to organize your images:

```
img/
├── profile.webp
├── post-name/
│   ├── image1.png
│   └── image2.jpg
└── another-post/
    └── diagram.svg
```

## Image Guidelines

- **Optimize images** before adding them to reduce file sizes
- **Use descriptive filenames** for better organization
- **Supported formats**: PNG, JPG, WebP, SVG, GIF
- **Recommended profile image size**: 400x400px or larger (square aspect ratio)
